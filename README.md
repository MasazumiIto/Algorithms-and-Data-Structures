# アルゴリズムとデータ構造
授業で使用していたディレクトリをそのまま上げました。
プログラムファイルの後半には出力結果が記述されているものがあります。
## 授業の説明
C言語を用いてハッシュテーブルや木構造、全探査アルゴリズムの実装を行います
## ファイルの説明
- init.txt
  kadai5.cで線形リストを作成するときに使用する駅のデータ
- unkonwn\d\d.txt  
  文字認識実験で使用する謎の文字の特徴量が記述されたファイル
- ave.dic  
  平仮名の平均特徴量が保存されているファイル
- eigenvv.dat  
  Kadai8.cで固有ベクトルを求めてこのファイルを作成し、Kadai9.cの認識実験で使用する
-cov.dic  
  Kadai8.cで固有値を求めてこのファイルを作成し、Kadai9.cの認識実験で使用する
- Kadai2.c  
  対象の文字列に特定の文字列が含まれているかどうか判定するプログラム
- Kadai3-1.c  
  線形探査と二分探索の実装を行ったプログラム
- kadai3-3.docx
  kadai2.cで実装したプログラムの効率化バージョンのプログラム
- Kadai4-1.c  
  空白や改行を除去するプログラム
- Kadai4.2.c  
  文字列型内の文字が全て数字であるかそうでないかを判定するプログラム
- kadai5.c
  線形リストを作成し、一連の操作を行うプログラム
- Kadai6-2.c  
  ハッシュテーブルを作成し、一連の操作を行うプログラム
- Kadai8.c  
  木構造を作成し、一連の操作を行うプログラム
- Kadai10.c  
  深さ優先探査を行うプログラム
- kadai11.c
  幅優先探査を行うプログラム
- kadai11-2.c
  幅優先探査を行うプログラム　その2
- kadai12.c
  ダイクストラ法を行うプログラム
- kadai13.c
  ワ―シャルフロイド法を行うプログラム
- kadai14.c
  ナップザック問題を解くプログラム
